# Project Name : AstroTalk

A responsive horoscope web application built using "HTML", "CSS", and "JavaScript". Users can explore horoscope predictions for all 12 zodiac signs across multiple time periods through an intuitive and interactive interface.

##  Key Features

* Zodiac Selection Grid

  * Responsive grid displaying all 12 zodiac signs with clear icons and labels.
  * Easy navigation for selecting any zodiac sign.

* Timeline Navigation

  * Interactive tabs for switching between:

    * Yesterday
    * Today
    * Tomorrow
    * Weekly
    * Monthly
  * Dynamically updates horoscope information based on the selected time period.

* Segmented Horoscope Insights

  * Displays horoscope details in organized sections:

    *  Career
    *  Love
    *  Health
    *  Finance
  * Uses structured layouts and visual indicators for better readability.

* Responsive Design

  * Optimized for desktop, tablet, and mobile devices.

* Vanilla JavaScript

  * Built without frameworks or libraries, using only HTML, CSS, and JavaScript.

## Local Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/SonuKumar86/AstroTalk.git
   ```

2. Navigate to the project folder:

   ```bash
   cd your-repository
   ```

3. Open `index.html` in your browser or use the "Live Server" extension in VS Code.
